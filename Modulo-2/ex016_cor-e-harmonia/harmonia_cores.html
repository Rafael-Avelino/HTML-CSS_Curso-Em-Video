Harmonia de cores

Cores primárias

As cores primárias são:

  • Amarelo
  • Azul
  • Vermelho

Elas possuem uma harmonia "grosseira".

Cores secundárias

As cores secundárias são:

  • Laranja
  • Violeta
  • Verde

Da mesma forma que as primárias, também apresentam harmonia grosseira.

Cores terciárias

As cores terciárias são todas as outras cores que sobraram do círculo cromático.

A cor terciária sempre vai estar entre uma cor primária e uma secundária. Dessa forma, a nomeclatura de uma cor terciária se dá pelo nome da cor primária seguido da cor secundária.

Sendo assim, as cores terciárias são:

  • Amarelo-esverdeado
  • Amarelo-alaranjado
  • Vermelho-alaranjado
  • Vermelho-arroxeado
  • Azul-arroxeado
  • Azul-esverdeado

Temperatura de cores

As cores também podem ser representadas por cores frias e quentes


Obs: Ao desenvolver um site, é preciso se preocupar com a paleta de cores. Uma paleta deve ter entre 3 e 5 cores, desconsiderando o branco e preto

Cores complementares

As cores complementares são as que tem mais contraste entre si.

Cores análogas

As cores análogas são cores que não apresentam tanto contraste, mas apresentam harmonia. São cores vizinhas dentro do círculo cromático

Cobinações de cores

Cores Análogas + Complementares

Uma boa forma de combinar cores é juntar cores análogas e complementares.

Essa forma cria um contraste muito forte ao adicionar a cor complementar.

Cores Análogas Relacionadas

2 cores vizinhas, pula a terceira cor e pega a quarta

Essa forma cria um contraste mais leve e agradável

Cores intercaladas

Pega uma cor, pula outra

Essa paleta cria um contraste mais duro

Monocromia

A mesma cor modificando saturação e luminosidade.

Paleta de cores

Uma ferramenta interessante para montar uma paleta de cores é o Adobe color.

Outra ferramenta é o Paletton. Nele é possível criar exemplos de site com a paleta escolhida.
