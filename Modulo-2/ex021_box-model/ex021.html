Modelo de caixas

Os modelos de caixa representam o layout de um elemento e são divididos entre box-level e inline-level.

Box-level

Os elementos box-level são elementos que quebram a continuidade do código para adicionar uma "caixa". Exemplo:

  • <div>
  • <h1>-<h6>
  • <p>
  • <main>
  • <header>
  • <nav>
  • <article>
  • <aside>
  • <footer>
  • <form>
  • <video>

Inline-level

Os elementos inline-level são elementos que não quebram a continuidade do código para adicionar uma "caixa". Exemplo:

  • <span>
  • <a>
  • <code>
  • <small>
  • <strong>
  • <em>
  • <sup>-<sub>
  • <label>
  • <button>
  • <input>
  • <select>
