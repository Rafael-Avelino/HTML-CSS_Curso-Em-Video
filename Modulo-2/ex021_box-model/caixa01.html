Exemplo de caixa box-level

Parágrafos também são exemplos de box-level, mas os links são exemplos de caixas inline-level. Vamos ver como tudo isso funciona.

Para melhor visualização, utilizar o dev tools do chrome

Nele é possível alterar as propriedades dos elementos sem que haja alteração do documento original

Anotações

Simplificando os comandos já vistos

Border

Antes:

        
    border-width: 10px;
    border-style: solid;
    border-color: #4E4B5C;

Depois:

        
    border: 10px solid #4E4B5C;

Padding

Antes:

        
    padding-top: 10px;
    padding-right: 10px;
    padding-bottom: 10px;
    padding-left: 10px;

Depois:

    
    padding: 10px 10px 10px 10px;

Obs.: se todos os tamanhos forem iguais, basta informar uma vez só:

padding: 10px;

Margin

Antes:

        
    margin-top: 10px;
    margin-right: 10px;
    margin-bottom: 10px;
    margin-left: 10px;

Depois:

    
    margin: 10px 10px 10px 10px;

Margin também pode ser configurada utilizando o valor "auto". Esse valor ajusta automaticamente a margin de acordo com o tamanho da viewport

Outline

Antes:

        
    outline-width: 5px;
    outline-style: dashed;
    outline-color: #0D0221;

Depois:

        
    outline: 5px dashed #0D0221;
