Anotações

Uma forma de personalizar elementos HTML é dando uma id ou uma class para eles. Utiliza-se "#" para id e "." para classes.

Adicionando id:

<h1 id="principal">HTML básico</h1>

Adicionando class:

<h2 class="intermed">HTML intermediário</h2>

A id só pode ser utilizada uma vez. Já a class pode ser utilizada em vários elementos HTML ao mesmo tempo.
