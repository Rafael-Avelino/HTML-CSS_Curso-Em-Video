Código-fonte e Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Aqui, o código foi envelopado com a tag <code>, a qual cria uma fonte mono-espaçada.

O <code> não organiza o código com identação. Exemplo:
let n = window.document.querySeclector("input#ntxt") let num = Number(n.value) if (num === 0){ window.alert("Zero") } else { window.alert("Diferente de zero") }

Para resolver isso, é só adicionar a tag <pre>: 

        
    let n = window.document.querySeclector("input#ntxt")
    let num = Number(n.value)

    if (num === 0){
        window.alert("Zero")
    } else {
        window.alert("Diferente de zero")
    }
