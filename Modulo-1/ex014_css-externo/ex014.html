Capítulo 1.1

O css é aplicado em um arquivo diferente de onde está o código HTML5.

Essa é a melhor forma pois permite estilizar várias páginas ao mesmo tempo ao copiar o código de link, exemplo:

        
            <link rel="stylesheet" href="style.css">

Neste exemplo as duas páginas possuem o mesmo link de CSS, portanto, apresentam o mesmo estilo
