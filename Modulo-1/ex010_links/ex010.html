Link para download

  • Link para download de Esteves 1998

Para criar um link para download como é necessário criar um link normal. Na href de <a> colocar o caminho do arquivo e adicionar também o atributo download e colocar o nome do arquivo e sua extensão.Exemplo:

<a> href="livro/Esteves_1998.pdf" download="Esteves_1998.pdf">Link para download de Esteves 1998</a>
