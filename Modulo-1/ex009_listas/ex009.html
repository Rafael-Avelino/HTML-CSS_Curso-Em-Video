Trabalhando com listas

OL (Ordered Lists) - Listas Ordenadas

A lista ordenada é demarcada pela tag <ol> e possui subtags <li> que são os list items.

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Ir pra faculdade
  5. Almoçar
  6. Estudar para bioquímica

Por padrão, a lista é ordenada por números. Porém, isso pode ser alterado utilizando "type" dentro da tag <ol>

UL (Unordered) - Listas não Ordenadas

A lista não ordenada é demarcada pela tag <ul> e possui subtags <li>

  • Arroz
  • Feijão
  • Salada
  • Macarrão

Por padrão, a lista é ordenada por discos.

Listas Mistas

Lista de Definições

A lista de definições é demarcada pela tag <dl>, dentro dela ficam as tags <dt> para o termo e <dd> para a definição.
