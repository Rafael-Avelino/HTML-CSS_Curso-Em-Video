Imagens dinâmicas

Tente abrir este site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho de seu navegador.

Neste exercício foi criada uma imagem dinâmica. As imagens foram colocadas dentro da tag <picture> e dentro dela foi colocada a imagem grande.

Para criar a dinâmica de tamanho foi colocada ainda dentro da tag <picture> a tag <source>. Dentro dessa tag é necessário colocar o tamanho máximo do navegador em pixels, a imagem desejada e o tipo da imagem.

        
    <picture>
        <source media="(max-width: 750px )" srcset="imagens/imagem-p.png" type="image/png">
        <source media="(max-width: 1050px )" srcset="imagens/imagem-m.png" type="image/png">
        <img src="imagens/imagem-g.png" alt="Imagem flexível">
    </picture>

A lógica envolvida para o código acima é a seguinte: se o navegador tiver menos que 750 pixels, carregue esta imagem.

Adicionando áudio

Aqui adicionaremos um audio através de HTML5:

A tag utilizada para adicionar áudio foi <audio>; dentro de "src" coloca-se o caminho do arquivo. Depois é necessário adicionar o parâmetro "controls" que mostra o player do áudio.

<audio src="midia/MerryGo_SilentFilmLight_KevinMacLeod.mp3" controls></audio>

Da forma indicada acima o audio pode não ter compatibilidade com alguns navegadores. Para corrigir isso, adicione outras extensões de mídia:

        
    <audio preload="metadata" controls>
    <source src="midia/MerryGo_SilentFilmLight_KevinMacLeod.mp3" type="audio/mpeg">
    <source src="midia/MerryGo_SilentFilmLight_KevinMacLeod.ogg" type="audio/ogg">
    <source src="midia/MerryGo_SilentFilmLight_KevinMacLeod.wav" type="audio/wav">
    </audio>

O parâmetro "preload" indica o que acontecerá com o áudio quando o site for carregado. Esse parâmetro apresenta três opções:

  • none - Indica que o áudio não deve ser pré carregado.
  • metadata - Indica que apenas os meta dados do áudio serão carregados
  • auto - Indica que o áudio pode ser carregado junto com a página.

Caso não seja indicado um "preload", o navegador entende como a opção "auto"

Adicionando vídeos

Este vídeo foi adicionado da seguinte forma:

        
    <video muted controls >
        <source src="midia/cachoeira_chapada-veadeiros.mp4" type="video/mp4">
        Seu navegador não suporta esse tipo de vídeo, mas não se preocupe, você
        pode <a href="midia/cachoeira_chapada-veadeiros.mp4" download="cachoeira_chapada-veadeiros.mp4">baixar ele aqui</a>
        e assistir em seu reprodutor de vídeo favorito.
    </video>

O parâmetro "muted" deixa o vídeo mudo por padrão, o cliente poderá desmutar o vídeo se quiser. Já o parâmetro controls torna visível os controles do vídeo (botão play/pause, mute, tela cheia e opções)
